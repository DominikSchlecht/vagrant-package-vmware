# vagrant-package-vmware
Python3-Script to back a VM from VMWare to a Vagrant Box.

# Usage
usage: vagrant-package-vmware.py [-h] [--vmdk VMDK]
                                 [--compression-level COMPRESSLEVEL]
                                 [--dest-dir DEST_DIR]
                                 vmx name version
